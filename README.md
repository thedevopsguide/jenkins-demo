# Jenkins Git Demo

This repository demonstrates a Jenkins Freestyle job integrated with GitHub.

## Tools Used
- Jenkins
- Git & GitHub
- Bash Shell

## What This Project Does
- Clones code from GitHub
- Runs a Jenkins Freestyle job
- Executes a Bash script
- Creates files and writes content automatically

## How to Run
1. Create a Jenkins Freestyle project
2. Configure GitHub repository URL
3. Add "Execute shell" build step
4. Run build.sh script
